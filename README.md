# 2026年 人事労務課職員旅行 思い出アルバム

## ファイル構成
- index.html / style.css / app.js …… アプリ本体(基本的に変更しません)
- album-data.js …… 写真・動画のリスト(写真を追加するたびに更新)
- photos/ …… 写真・動画ファイルを入れるフォルダ

## 写真を追加する手順
1. Claudeのチャットに写真・動画を添付する
2. AIが内容を解析し、更新済みの album-data.js を作成
3. GitHubで photos/ フォルダに写真をアップロード
4. album-data.js を上書きアップロード
以上で、ページ構成・並び順は自動で最適化されます。
